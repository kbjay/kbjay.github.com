JVM - jvm工具

jvm工具简介

命令行

  1. jps（查看jvm进程id）

  2. jstat （查看jvm进程的运行时内存状态）

    对应的信息。survivor0，1，eden，老年代，元数据，压缩类信息，新生代gc次数，新生代gc时间。。。

  3. jinfo （查看jvm配置）

  4. jstack （查看线程运行信息）

  5. jMap (堆转储)

可视化

  1. jconsole （查看cpu，内存，线程等）

  2. VisualVm （多和一，想看啥看啥，支持安装插件）
